Методология проекта

Архитектура системы, основные компоненты и используемые технологии

Проект SAWrap реализован как веб-ориентированная система для сравнения моделей классификации, регрессии и анализа выживаемости. Методология проекта основана на разделении пользовательского интерфейса, вычислительного модуля и слоя хранения результатов, что позволяет обеспечить воспроизводимость экспериментов, удобный доступ к результатам через сайт и простоту развёртывания на сервере.

Компоненты системы

  • GitHub-репозиторий с исходным кодом проекта и конфигурацией деплоя: github.com/DimonZhi/SAWrap
  • Веб-сайт на FastAPI для взаимодействия пользователя с системой и визуализации результатов.
  • Вычислительный модуль на Python для запуска экспериментов, расчёта метрик и сравнения моделей.
  • Хранение предрассчитанных результатов в xlsx-таблицах для ускорения работы интерфейса и повторного использования результатов.

Используемые технологии

  • Backend и интерфейс: Python, FastAPI, Jinja2, Uvicorn.
  • Машинное обучение и survival analysis: scikit-learn, lifelines, scikit-survival, survivors.
  • Обработка данных и визуализация: Pandas, NumPy, SciPy, Plotly.
  • Инфраструктура и публикация: GitHub, Docker, Nginx.
